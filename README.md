# Cathay Award Flight Finder

Chrome extension for adding an award flight finder panel to Cathay Pacific booking pages.

## Features

- Adds a search panel on supported Cathay Pacific pages.
- Saves the user's last entered route, date, passenger count, and language preference locally in the browser.
- Supports batch award availability searches across dates and comma-separated airport codes.
- Displays available cabins, flight details, and estimated Cathay points where the data is available.

## Install for local testing

1. Open `chrome://extensions/`.
2. Enable Developer mode.
3. Choose "Load unpacked".
4. Select this folder.

## Store package

Zip the extension folder contents for Chrome Web Store upload. The package should include `manifest.json`, `content.js`, `icons/`, `README.md`, and `PRIVACY.md`.
