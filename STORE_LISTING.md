# Chrome Web Store Listing Draft

## Name

Cathay Award Flight Finder

## Short description

Adds an award flight finder panel to Cathay Pacific booking pages.

## Detailed description

Cathay Award Flight Finder helps users search award flight availability from supported Cathay Pacific booking pages.

Features:

- Search award availability from a compact panel on Cathay Pacific pages.
- Run batch searches across multiple dates.
- Enter comma-separated airport codes to compare route combinations.
- Filter displayed results by cabin and non-stop flights.
- View flight numbers, departure and arrival times, transit details, and estimated Cathay points where available.

The extension stores search preferences locally in the browser and communicates only with Cathay Pacific domains needed for the search workflow.

## Single purpose

Help users search and compare Cathay Pacific award flight availability from supported Cathay Pacific booking pages.

## Permission justification

Host permissions for `cathaypacific.com` and `book.cathaypacific.com` are required so the extension can add the search panel, read booking-page state, and submit user-requested award availability searches to Cathay Pacific pages.
